# Project Title  
**Lab 2: Control Statements and Data Handling using Python**

## Description  
The objective of this project is to understand and implement data handling techniques in Python, including loading datasets from different file formats, performing basic data exploration, and visualizing key insights using graphical methods.

## Dataset  
- **Source:** Local files (CSV, TXT, Excel)  
- **Description of data:**  
  The dataset contains information about Facebook friends, including attributes such as age, number of friends, and gender. It is used to demonstrate file handling, data cleaning, and visualization techniques.

## Steps Performed  
1. **Data Cleaning**  
   - Loaded data from CSV, TXT, and Excel files  
   - Handled encoding issues in TXT file  
   - Cleaned column names (removed extra spaces)  
   - Checked for missing values  

2. **Exploratory Data Analysis**  
   - Displayed dataset structure using `.head()` and `.info()`  
   - Analyzed dataset shape and column names  
   - Generated summary statistics using `.describe()`  

3. **Visualization**  
   - Created histogram for Age distribution  
   - Created histogram for Number of Friends  
   - Generated count plot for Gender distribution  

4. **Model Building (if applicable)**  
   - Not applicable for this lab  

## Results  
- **Key findings:**  
  - Age distribution shows variation among users  
  - Number of friends varies significantly across individuals  
  - Gender distribution provides insight into dataset composition  

- **Metrics (if applicable):**  
  - Not applicable  

## Tools Used  
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  

## Conclusion  
This project demonstrates how to load and analyze datasets from multiple formats and perform basic exploratory data analysis. Visualization techniques help in understanding patterns and distributions within the data effectively.
