
# Number Guessing Game

## Description

This project is a fun and interactive Number Guessing Game developed using Python. The objective of the game is for the user to guess a randomly generated number within a limited number of attempts.

## Dataset

* Source: Not applicable
* Description: This project does not use a dataset. The number is randomly generated using Python's random module.

## Steps Performed

1. Generated a random number between 1 and 100
2. Took user input for guesses
3. Compared the input with the generated number
4. Provided feedback such as "too high" or "too low"
5. Implemented a hint system to indicate closeness
6. Added score tracking and attempt limits
7. Displayed results using ASCII art
8. Included a replay feature for multiple plays

## Results

* Successfully created an interactive game
* Implemented scoring and hint system
* Improved user experience with visual elements and feedback

## Tools Used

* Python
* random
* time

## Conclusion

This project demonstrates the use of basic Python concepts such as loops, conditional statements, user input, and random number generation in a creative and interactive way.
