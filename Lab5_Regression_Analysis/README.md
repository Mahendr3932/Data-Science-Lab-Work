# Lab 5: Regression Analysis using Linear Regression

## Description
This lab demonstrates the implementation of a simple linear regression model to predict outcomes based on input features.

## Dataset
- Source: Manually created dataset
- Description of data:
  The dataset represents the relationship between hours studied and marks obtained.

## Steps Performed
1. Data Cleaning  
   - Dataset structured and clean

2. Exploratory Data Analysis  
   - Relationship between study hours and marks observed

3. Visualization  
   - Scatter plot and regression line plotted

4. Model Building (if applicable)  
   - Data split into training and testing sets  
   - Linear Regression model trained  
   - Predictions made on test data

## Results
- Model predicted marks based on study hours  
- Mean Squared Error calculated  
- R² Score evaluated model performance  

## Tools Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Scikit-learn  

## Conclusion
This lab demonstrated how regression models can be used for prediction tasks and how model performance is evaluated using standard metrics.
